# Video Tutorial Script: Taking Your First Quiz

**Target Audience**: Students  
**Duration**: 5-7 minutes  
**Goal**: Demonstrate how to access, take, and complete a quiz with all question types

---

## Introduction (30 seconds)

**On Screen**: Open-Edu dashboard homepage

**Voiceover**:
"Welcome to Open-Edu! In this tutorial, you'll learn how to take your first quiz. We'll walk through finding a quiz, answering different question types, and viewing your results. Let's get started!"

---

## Part 1: Accessing a Quiz (1 minute)

**On Screen**: 
1. Dashboard showing enrolled courses
2. Click on "Introduction to Web Development" course card
3. Navigate to course detail page

**Voiceover**:
"From your dashboard, click on any course you're enrolled in. I'm selecting 'Introduction to Web Development.' On the course page, you'll see all available lessons and quizzes."

**On Screen**:
4. Scroll to show lesson list

"Quizzes are marked with a purple badge and quiz icon. Click 'Start Quiz' to begin. If you've already attempted it, you'll see 'Retake Quiz' instead."

## Post-Production Notes

### Visual Highlights
- Use circles or arrows to point out UI elements (timer, progress dots, buttons)
- Add zoom-ins for important actions (clicking buttons, selecting answers)
- Use color highlights (yellow or red boxes) for critical information

### Editing Tips
- Add subtle background music (keep it low volume)
- Include text overlays for key points:
  - "Multiple Choice: Select ONE answer"
  - "Multiple Answer: Select ALL correct answers"
  - "Watch the timer!"
- Speed up repetitive actions (like scrolling) slightly

### Accessibility
- Provide closed captions for all voiceover
- Ensure high contrast in video quality
- Use clear, readable fonts for on-screen text
