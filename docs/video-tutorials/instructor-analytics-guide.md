## Post-Production Notes

### Visual Highlights
- Use colored boxes to highlight key metrics (green for good, yellow for warning, red for at-risk)
- Add arrows pointing to important data points
- Zoom in on specific numbers when discussing them
- Use side-by-side comparisons for before/after scenarios

### Editing Tips
- Add subtle transitions between tab switches
- Include "pop-up" animations for key insights
- Speed up scrolling through long lists
- Add sound effects for notifications or alerts

### On-Screen Graphics
- Create simple bar charts or graphs to visualize trends
- Add text callouts for key takeaways:
  - "52% completion = Red Flag"
  - "45% correct = Review this question"
  - "3 at-risk students = Time to reach out"

### Charts to Include (if not in actual UI)
- Progress distribution histogram
- Engagement trend line over time
- Completion rate by lesson (bar chart)
- Average quiz scores by assessment (line graph)

### Accessibility
- Provide closed captions for all voiceover
- Describe visual charts verbally for audio-only viewers
- Use high-contrast colors for data visualization
- Ensure all on-screen text is readable (18pt minimum for data)
- Describe color coding ("green indicates above 70%, red below 50%")

### Call-to-Action
- End with prompt to check their own course analytics
- Suggest scheduling regular analytics reviews
- Encourage sharing insights with teaching teams
