## Starting a Lesson

### Accessing Course Content

1. **From Your Dashboard**
   - Click on the course card
   - Click "Continue Learning" to resume
   - Or click "View Course" for the full overview

2. **From the Course Page**
   - View the complete lesson list
   - Click any lesson title to start
   - Lessons show completion status (✓ for completed)

### Lesson Page Layout

When you open a lesson, you'll see:

```
┌─────────────────────────────────────────────────┐
│ [Course Title] > [Lesson Title]  [Auto-saved ✓] │ <- Top Bar
├──────────┬──────────────────────────────────────┤
│ Sidebar  │  Main Content Area                   │
│          │                                       │
│ Lesson 1 │  # Lesson Heading                    │
│ ✓ Intro  │                                       │
│          │  Reading content here...             │
│ Lesson 2 │                                       │
│ → Current│  Scroll to read                      │
│          │                                       │
│ Lesson 3 │                                       │
│ [ ] Next │                                       │
├──────────┴──────────────────────────────────────┤
│  [← Previous]        [Next Lesson →]            │ <- Navigation
└─────────────────────────────────────────────────┘
```

## Reading Content

### Content Types

Open-EDU lessons support rich markdown formatting:

1. **Text & Paragraphs**
   - Regular text
   - **Bold** and *italic* formatting
   - Links to external resources

2. **Headings & Structure**
   - # Main Headings
   - ## Subheadings
   - Organized, hierarchical content

3. **Lists**
   - Bullet points
   - Numbered lists
   - Nested lists

4. **Code Examples**
   ```javascript
   // Syntax-highlighted code
   function example() {
     return "Learn by doing!";
   }
   ```

5. **Images & Media**
   - Embedded images
   - Diagrams and charts
   - Video embeds (if supported)

6. **Tables**
   - Structured data presentation
   - Comparison tables

### Reading Progress

Your reading progress is automatically tracked:

- **Scroll Position**: Saved as you scroll
- **Visual Indicator**: Progress bar at the top of the content
- **Auto-Save**: Your position is saved every few seconds
- **Resume**: Return exactly where you left off

> **Note**: You'll see "Saving..." or "Saved ✓" indicator in the top-right corner

### Reading Modes

**Focus Mode**
- Hide the sidebar for distraction-free reading
- Toggle with `Cmd+S` (Mac) or `Ctrl+S` (Windows)
- Content centers on the screen
- Press again or `Esc` to restore sidebar

**Reading Mode Toggle**
- Adjust text size and spacing
- Optimized for comfortable reading
- Look for the reading mode button in the toolbar

## Taking Notes

Notes help you capture insights, questions, and key takeaways as you learn.

### Opening the Notes Panel

1. Click the **"Notes"** button (usually at the bottom or in the toolbar)
2. The notes panel slides up from the bottom
3. You'll see any existing notes for this lesson

### Creating a Note

1. Type your note in the text area
2. Notes are automatically saved as you type
3. You'll see a "Saving..." indicator while saving
4. No need to manually save - it's automatic!

### Note Features

**Markdown Support**
- Format your notes with markdown
- Use `#` for headings
- Use `-` for bullet points
- Use `` ` `` for inline code

**Per-Lesson Notes**
- Each lesson has its own notes section
- Notes are private to you
- Access them anytime you revisit the lesson

**Search & Review**
- Review notes later from your profile
- Search across all your notes (if available)
- Export notes for offline study

### Best Practices for Note-Taking

1. **Summarize in Your Own Words**: Don't copy-paste - rephrase concepts
2. **Highlight Key Concepts**: Use bold or headings for important points
3. **Ask Questions**: Note things you want to explore further
4. **Create Examples**: Add your own examples to reinforce learning
5. **Review Regularly**: Revisit notes before quizzes or exams

## Using Bookmarks

Bookmarks let you quickly return to important lessons.

### Adding a Bookmark

1. Open the lesson you want to bookmark
2. Click the **Bookmark** button (⭐ or bookmark icon)
3. The lesson is added to your bookmarks
4. Button changes to indicate it's bookmarked

### Viewing Bookmarked Lessons

**From Your Profile**:
1. Click your avatar → **Profile**
2. Navigate to the **Bookmarks** section
3. See all bookmarked lessons across courses

**From the Course Page**:
- Bookmarked lessons may have a special indicator
- Filter to show only bookmarked lessons (if available)

### Removing a Bookmark

1. Open the bookmarked lesson
2. Click the bookmark button again
3. Or remove from your bookmarks list in your profile

### When to Use Bookmarks

- Key lessons you want to review before a quiz
- Complex topics that need revisiting
- Reference lessons with important formulas or code
- Lessons with external resources you might need later

## Tracking Your Progress

Open-EDU automatically tracks your learning progress across multiple dimensions.

### Progress Indicators

1. **Lesson Completion**
   - ✓ Green checkmark = Completed
   - → Arrow = Currently in progress
   - Empty checkbox = Not started

2. **Course Progress**
   - Percentage bar on the course card
   - Shows overall completion (e.g., "45% complete")
   - Based on lessons marked as complete

3. **Reading Progress**
   - Per-lesson progress indicator
   - Shows how much you've scrolled/read
   - Updates in real-time as you read

### How Lessons Are Marked Complete

**Automatic Completion**:
- Scroll to the bottom of a text lesson
- Complete and submit a quiz (for quiz lessons)
- Progress automatically saved

**Manual Completion**:
- Some lessons have a "Mark as Complete" button
- Click when you've finished the content
- Useful if you revisit a lesson

### Viewing Your Progress

**Dashboard Overview**:
- Overall progress across all courses
- Recent activity timeline
- Completion statistics

**Course Page**:
- Progress per lesson
- Overall course completion percentage
- Estimated time remaining

**Profile Page**:
- Full learning history
- Certificates earned (if available)
- Total time spent learning

## Navigation Features

### Sidebar Navigation

The left sidebar shows:
- **All lessons in the course**
- **Completion status** for each
- **Current lesson** highlighted
- **Search box** to find specific lessons

**Sidebar Tips**:
- Click any lesson to jump directly to it
- Use search to filter lessons by name
- Collapse/expand lesson groups (if applicable)
- Hide sidebar with `Cmd+S` / `Ctrl+S` for focus mode

### Search Lessons

1. Press `/` or click in the search box
2. Type keywords to filter lessons
3. Results update as you type
4. Click a result to navigate
5. Press `Esc` or click X to clear search

### Quick Navigation Buttons

**Bottom Navigation Bar**:
- **Previous Lesson**: Go to the previous lesson in sequence
- **Next Lesson**: Move to the next lesson
- Disabled if you're at the first/last lesson

**Keyboard Shortcuts**:
| Shortcut | Action |
|----------|--------|
| `←` | Previous lesson |
| `→` | Next lesson |
| `/` | Search lessons |
| `Cmd+S` / `Ctrl+S` | Toggle focus mode |
| `Esc` | Close panels/modals |

### Breadcrumb Navigation

At the top of the page:
- **[Course Title]** → **[Lesson Title]**
- Click the course title to return to course overview
- Always know where you are in the course

## Study Tips & Best Practices

### Effective Learning Strategies

1. **Set a Regular Schedule**
   - Dedicate specific times for studying
   - Consistency beats long, irregular sessions
   - Even 20-30 minutes daily makes a difference

2. **Active Learning**
   - Take notes in your own words
   - Create examples beyond what's provided
   - Explain concepts to yourself out loud

3. **Spaced Repetition**
   - Review lessons after 1 day, 1 week, 1 month
   - Use bookmarks for important review material
   - Revisit quizzes to reinforce concepts

4. **Focused Study Sessions**
   - Use Focus Mode to minimize distractions
   - Turn off phone notifications
   - Close unrelated browser tabs
   - Set a timer for focused work periods

5. **Ask Questions**
   - Note questions in the notes section
   - Reach out to your instructor
   - Discuss with classmates (if applicable)

### Using Auto-Save Effectively

- Auto-save means you can safely close the browser
- Always resume where you left off
- No need to click "Save" - it's automatic
- Watch for the "Saved ✓" indicator for peace of mind

### Progress Tracking Best Practices

- Check your dashboard weekly to monitor progress
- Set goals: "Complete 3 lessons this week"
- Celebrate milestones: 25%, 50%, 75%, 100%
- Don't skip lessons - they build on each other

### Mobile Learning

Open-EDU works great on mobile devices:
- Touch-friendly navigation
- Responsive reading layout
- Notes and bookmarks work everywhere
- Progress syncs across all your devices

## Troubleshooting Common Issues

### Lesson Won't Mark as Complete

**Solutions**:
- Scroll to the very bottom of the lesson
- Look for a "Mark as Complete" button and click it
- Refresh the page and check again
- If issue persists, see [Troubleshooting Guide](./troubleshooting.md)

### Can't Find My Notes

**Solutions**:
- Notes are per-lesson - make sure you're on the right lesson
- Check your internet connection
- Refresh the page to reload notes
- Check your profile for archived notes

### Progress Not Saving

**Solutions**:
- Check for "Saved ✓" indicator
- If you see "Saving..." for too long, check your internet
- Try refreshing the page
- Clear browser cache and retry
- See [Troubleshooting Guide](./troubleshooting.md) for more help
