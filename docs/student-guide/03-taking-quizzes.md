## Quiz Basics

### What Are Quizzes?

Quizzes are assessments that test your understanding of course material. They can be:
- **Formative**: Practice quizzes to check understanding
- **Summative**: Graded assessments that count toward completion
- **Self-paced**: Take at your own pace
- **Timed**: Complete within a time limit

### Quiz Settings

Each quiz may have different settings:
- **Time Limit**: Some quizzes are timed (e.g., 30 minutes)
- **Attempts Allowed**: Number of times you can take the quiz
- **Passing Score**: Minimum score required to pass (often 70%)
- **Randomization**: Question or answer order may be randomized
- **Feedback**: Immediate or delayed results

## Starting a Quiz

### Locating Quiz Lessons

1. Open your course
2. Look for quiz lessons in the lesson list
3. Quiz lessons are typically labeled with a quiz icon or badge
4. Click the quiz lesson to open it

### Quiz Introduction Page

Before starting, you'll see:
- **Quiz Title** and description
- **Number of Questions** (e.g., "10 questions")
- **Time Limit** (if timed, e.g., "30 minutes")
- **Passing Score** (e.g., "70% to pass")
- **Attempts Remaining** (e.g., "2 of 3 attempts left")
- **Previous Scores** (if you've taken it before)

### Start Quiz Button

1. Read the quiz information carefully
2. Make sure you have enough time to complete it
3. Click **"Start Quiz"** when ready
4. The timer starts immediately (for timed quizzes)

> **Important**: Once you start, the timer begins. Don't start unless you're ready to complete it!

## Question Types

Open-EDU supports 6 different question types. Here's how to answer each:

### 1. Multiple Choice

**Format**: One correct answer from several options

**How to Answer**:
- Click the radio button next to your answer
- Only one option can be selected
- Your selection highlights
- Click again to deselect (before submitting)

**Example**:
```
What is 2 + 2?
○ 3
● 4  ← Selected
○ 5
○ 6
```

### 2. Multiple Answer (Checkboxes)

**Format**: Select ALL correct answers

**How to Answer**:
- Click checkboxes next to all correct answers
- Multiple selections allowed
- Check all that apply
- You must select ALL correct answers to get points

**Example**:
```
Which of these are programming languages?
☑ Python   ← Selected
☐ HTML
☑ Java     ← Selected
☐ CSS
```

### 3. True/False

**Format**: Statement is either true or false

**How to Answer**:
- Click "True" or "False"
- Only two options
- Simple binary choice

**Example**:
```
The Earth orbits the Sun.
● True  ← Selected
○ False
```

### 4. Short Answer (Text Input)

**Format**: Type a brief answer (usually one word or phrase)

**How to Answer**:
- Type your answer in the text field
- Keep it concise and accurate
- Spelling may matter (check instructions)
- Case sensitivity varies by instructor

**Example**:
```
What is the capital of France?
[Paris          ]  ← Type here
```

**Tips**:
- Check for spelling
- Avoid extra spaces
- Follow any format requirements (e.g., "YYYY-MM-DD")

### 5. Long Answer (Text Area)

**Format**: Write a paragraph or essay response

**How to Answer**:
- Type your answer in the larger text area
- Multiple sentences or paragraphs allowed
- May support markdown formatting
- Graded manually by instructor

**Example**:
```
Explain the water cycle:
┌─────────────────────────────────┐
│ The water cycle consists of...  │
│                                  │
│ [Your essay here]                │
│                                  │
└─────────────────────────────────┘
```

**Tips**:
- Structure your answer with paragraphs
- Use clear, concise language
- Proofread before submitting
- These are often manually graded

### 6. Matching

**Format**: Match items from Column A with items from Column B

**How to Answer**:
- Use dropdown menus to select matches
- Each item in Column A pairs with one item in Column B
- Make sure all items are matched

**Example**:
```
Match the country with its capital:
France     → [Paris ▼]
Germany    → [Berlin ▼]
Italy      → [Rome ▼]
```

## Taking a Timed Quiz

### Understanding the Timer

**Timer Display**:
- Always visible at the top of the quiz
- Shows time remaining (e.g., "24:35")
- Updates in real-time

**Color Indicators**:
- **Green**: More than 5 minutes remaining
- **Yellow**: 1-5 minutes remaining (warning)
- **Red**: Less than 1 minute remaining (urgent!)

**Visual Progress**:
- Circular progress indicator
- Shows percentage of time remaining
- May pulse when time is low

### Time Warnings

You'll receive visual warnings at:
1. **5 Minutes Remaining**: Color changes to yellow
2. **1 Minute Remaining**: Color changes to red, may pulse
3. **Time's Up!**: Automatic submission

> **Auto-Submit**: When time runs out, your quiz is automatically submitted with all answered questions. Unanswered questions score 0 points.

### Managing Your Time

**Best Practices**:
1. **Budget Your Time**: Divide time by number of questions
   - Example: 30 minutes ÷ 10 questions = 3 minutes per question
2. **Answer Easy Questions First**: Build confidence and save time
3. **Flag Difficult Questions**: Come back to them later
4. **Leave Time to Review**: Reserve 2-5 minutes at the end
5. **Don't Panic**: Yellow/red timer is just a warning

## Answering Questions

### Question Navigation

**Question Indicators**:
- Numbers for each question (e.g., "Question 1 of 10")
- Current question highlighted
- Quick navigation grid available

**Navigation Buttons**:
- **Previous**: Go back to the previous question
- **Next**: Move to the next question
- **Question Grid**: Click any question number to jump to it

### Question States

Questions can be in different states:
- **Answered** (✓): You've selected/typed an answer
- **Unanswered** (empty): No answer selected yet
- **Current** (highlighted): Question you're on now

### Changing Your Answers

**Before Submission**:
- Change answers as many times as you want
- Navigate back to any question
- Your latest answer is saved

**After Submission**:
- Answers are locked
- You cannot change them
- You can review them in results

### Mobile Quiz Experience

On mobile devices:
- Touch-friendly answer buttons (larger targets)
- Swipe gestures may be supported
- Timer stays visible at the top
- Bottom navigation bar for easy access
- Responsive layout optimized for small screens

## Submitting Your Quiz

### Before You Submit

**Final Checklist**:
- [ ] All questions answered (check for unanswered questions)
- [ ] Reviewed your answers (if time permits)
- [ ] Double-checked tricky questions
- [ ] No blank responses (unless intentional)

### Submit Quiz Button

1. When ready, click **"Submit Quiz"**
2. You'll see a confirmation dialog:
   ```
   Are you sure you want to submit?
   
   Answered: 9 of 10 questions
   Time Remaining: 5:23
   
   [Cancel]  [Submit Quiz]
   ```
3. Review the summary
4. Click **"Submit Quiz"** to confirm

> **Warning**: Once submitted, you cannot change your answers. Make sure you're ready!

### What Happens After Submission

Depending on quiz settings:
1. **Immediate Results**: See your score right away
2. **Delayed Results**: Instructor releases scores later
3. **Manual Grading**: Long answers graded by instructor

## Reviewing Results

### Quiz Results Page

After submission (if immediate feedback is enabled), you'll see:

**Score Summary**:
- **Your Score**: Percentage (e.g., "85%")
- **Points**: Raw score (e.g., "17/20 points")
- **Status**: Pass or Fail (with indicator)
- **Time Taken**: How long you spent
- **Attempt Number**: Which attempt this was

**Pass/Fail Indicator**:
- **✓ Passed**: Green banner, trophy icon
- **✗ Failed**: Red banner, retry encouragement

### Reviewing Individual Questions

**Question-by-Question Review**:
- See each question with your answer
- **Correct answers**: Green with ✓
- **Incorrect answers**: Red with ✗
- **Correct answer shown**: The right answer is displayed
- **Explanations**: Instructors may provide explanations

**Example**:
```
Question 1: What is 2 + 2?
Your Answer: 4 ✓ Correct

Question 2: What is the capital of Germany?
Your Answer: Munich ✗ Incorrect
Correct Answer: Berlin
Explanation: Berlin is the capital and largest city of Germany.
```

### Understanding Your Score

**Scoring Breakdown**:
- Each question is worth a certain number of points
- Correct answers earn full points
- Incorrect answers earn 0 points (usually)
- Partial credit may be available for some question types

**Passing Score**:
- Set by your instructor (often 70%)
- You must meet or exceed this to pass
- Failing allows retakes (if attempts remain)

## Retaking Quizzes

### Attempts Allowed

**Attempt Limits**:
- Set by instructor (e.g., "3 attempts")
- Check "Attempts Remaining" before starting
- Each attempt is tracked separately

### Viewing Attempt History

**Attempt History Section**:
1. On the quiz page, look for **"Attempt History"**
2. See all your previous attempts:
   - Attempt number
   - Score achieved
   - Date and time
   - Duration
   - **Best Score** highlighted with trophy icon

**Example**:
```
Your Quiz Attempts
─────────────────────────────────────
Attempt 3 • 85% ⭐ Best Score
Nov 20, 2025 • 2:15 PM • 28 min
[View Results]

Attempt 2 • 70% • Passed
Nov 19, 2025 • 4:30 PM • 25 min
[View Results]

Attempt 1 • 65% • Failed
Nov 18, 2025 • 3:00 PM • 30 min
[View Results]
```

### Retake Strategy

**Between Attempts**:
1. **Review Your Results**: Identify weak areas
2. **Study Missed Concepts**: Revisit lesson material
3. **Take Notes**: Focus on areas you got wrong
4. **Wait a Bit**: Give yourself time to absorb material
5. **Retake When Ready**: Don't rush your next attempt

**During Retakes**:
- Questions may be randomized (different order)
- Question pool may vary (different questions)
- Same time limit applies
- Previous answers are NOT pre-filled

## Sharing Your Achievements

### Certificate Generation (For Passed Quizzes)

When you pass a quiz, you can share your achievement!

**Share Certificate Button**:
- Appears on the quiz results page
- Only available for passed quizzes
- Click to open sharing options

**Sharing Options**:

1. **Download HTML Certificate**
   - Beautiful, printable certificate
   - Includes your name, quiz name, score, date
   - Professional design with gradients
   - Opens in a new tab for printing

2. **Copy Text Certificate**
   - ASCII art certificate
   - Copy to clipboard
   - Paste in emails, messages, or documents
   - Example format:
     ```
     ╔════════════════════════════════════╗
     ║    CERTIFICATE OF COMPLETION      ║
     ║                                    ║
     ║          [Your Name]               ║
     ║                                    ║
     ║  Has successfully completed        ║
     ║    [Quiz Name]                     ║
     ║                                    ║
     ║  Score: 85% • Passed ✓             ║
     ║  Date: November 20, 2025           ║
     ╚════════════════════════════════════╝
     ```

3. **Copy Share Link**
   - Generate a shareable URL
   - Share on social media
   - Include score and quiz details
   - Copy to clipboard with one click

**Feedback**:
- "Copied to clipboard!" message appears
- Dropdown closes automatically
- Certificate can be generated multiple times

## Tips for Quiz Success

### Before the Quiz

1. **Review Course Material**
   - Read all lessons thoroughly
   - Review your notes
   - Revisit bookmarked lessons

2. **Practice First**
   - If practice quizzes are available, take them
   - Get comfortable with the interface
   - Learn the question types

3. **Prepare Your Environment**
   - Find a quiet place
   - Ensure stable internet connection
   - Charge your device (if on mobile)
   - Close distracting tabs/apps

4. **Understand Quiz Settings**
   - Check time limit
   - Note passing score
   - Know how many attempts you have
   - Read any special instructions

### During the Quiz

1. **Read Carefully**
   - Read each question thoroughly
   - Watch for keywords: "NOT", "EXCEPT", "ALWAYS"
   - Understand what's being asked

2. **Answer Strategically**
   - Answer easy questions first
   - Skip difficult questions, return later
   - Budget your time wisely
   - Don't spend too long on one question

3. **For Multiple Answer Questions**
   - Treat each option as a true/false question
   - You must select ALL correct answers
   - Missing one correct answer = incorrect

4. **For Text Answers**
   - Be concise and clear
   - Check spelling
   - Answer the question directly
   - Avoid unnecessary information

5. **Watch the Timer**
   - Keep an eye on remaining time
   - Don't panic when it turns yellow/red
   - Reserve time to review
   - Better to submit with time left than let it expire

6. **Review Before Submitting**
   - Check for unanswered questions
   - Verify you didn't misclick
   - Double-check calculations
   - Ensure text answers are complete

### After the Quiz

1. **Review Results Carefully**
   - Understand why you got answers wrong
   - Read explanations provided
   - Note patterns in mistakes

2. **Learn from Mistakes**
   - Study concepts you missed
   - Revisit relevant lessons
   - Ask instructor for clarification

3. **Plan Your Retake** (if needed)
   - Don't immediately retake
   - Give yourself time to study
   - Focus on weak areas
   - Approach with fresh perspective

4. **Celebrate Success**
   - Share your certificate
   - Mark your achievement
   - Move on to next lesson

## Common Quiz Issues

### "Time's Up" Modal Appears

**What it means**: Your time expired
**What happens**: Quiz auto-submits with current answers
**Prevention**: Manage time better, watch the timer

### Can't Select an Answer

**Possible causes**:
- Internet connectivity issue
- Browser compatibility
- JavaScript disabled

**Solutions**:
- Refresh the page (quiz should resume)
- Try a different browser
- Check internet connection
- Contact instructor if issue persists

### Lost Progress After Closing Browser

**Solution**:
- Answers should be saved automatically
- Return to quiz and resume
- If not saved, you may need to restart
- Contact support if this happens frequently

### Results Not Showing

**Possible reasons**:
- Instructor set delayed feedback
- Manual grading required (for essay questions)
- Quiz is still being submitted

**Solutions**:
- Wait for instructor to release scores
- Check back later
- Contact instructor for timeline
